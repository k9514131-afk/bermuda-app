# 🚀 بدء سريع - Quick Start Guide

## الطرق السريعة للتشغيل

### 1️⃣ عبر VS Code (الطريقة الموصى بها)
اضغط `F5` واختر:
- **Full Stack (Next.js + Laravel)** - تشغيل الاثنين معاً ⭐
- **Frontend (Next.js)** - تشغيل الواجهة فقط
- **Backend (Laravel)** - تشغيل الخلفية فقط

### 2️⃣ عبر npm commands
```bash
# تشغيل الواجهة مع فتح المتصفح تلقائياً
npm run dev:open

# تشغيل الخلفية
npm run backend
```

### 3️⃣ عبر VS Code Tasks
- اضغط `Ctrl+Shift+B` أو `Cmd+Shift+B`
- اختر المهمة المطلوبة من القائمة

## 📊 المسارات والموارد

### الواجهة الأمامية (Next.js)
- **العنوان:** `http://localhost:9002`
- **المجلد:** `src/app`

### الخلفية (Laravel)
- **العنوان:** `http://127.0.0.1:8000`
- **API مثال:** `http://127.0.0.1:8000/api/rooms`
- **المجلد:** `laravel-backend`

### قاعدة البيانات (MySQL)
- **الاسم:** `bermuda_db`
- **ملف SQL:** `laravel-backend/database_export.sql`
